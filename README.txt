Конфигурация

    $uploadPath - путь к папке с хранимыми файлами
    $maxFileSize -  Максимальный размер принимаемых файлов. Указывать в байтах
    $hostName - адрес сайта  
    $dbHost - имя хоста для базы данных
    $dbUser - имя пользователя базы данных
    $dbPassword - пароль к базе данных
    $dbName - имя базы данных
    $templatesPath = 'uppy/templates/';
